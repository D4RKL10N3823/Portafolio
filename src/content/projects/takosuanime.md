title: TakosuAnime
desc: Plataforma web Full Stack para la gestión y visualización de catálogos de anime.
challenge: Construir una aplicación web monolítica robusta que integrara gestión de usuarios segura (con recuperación de contraseñas y ReCAPTCHA), administración de archivos multimedia y navegación dinámica de contenidos.
solution: Implementé una arquitectura MTV (Model-Template-View) utilizando Django. Diseñé modelos relacionales para series y episodios, integré el sistema de autenticación nativo con validaciones personalizadas y desarrollé un frontend responsivo mediante el sistema de plantillas de Django.
